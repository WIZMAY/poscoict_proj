# Posco ICT 인턴 과제
#  1.개요 
Kiosk 프로그램과 Level2 서버 프로그램의 기능을 구현하고 tcp 연결   
#  2.개발환경
OS : window 10   
개발언어 : C#   
DB : MySQL 8.0.30   
Tool : visual studio 2022 community (17.3.3)   
외부 라이브러리 : zbar(바코드 인식&생성), corescanner(zebra 홈페이지 제공)
